Katana Web Viewer

Katana also included a web app that was used to give end users the ability to upload and create their own work instructions and tutorials that could be viewed using Katana XR. It quickly became apparent that we needed a way to show the users 3D assets that had been uploaded before attaching them to tutorials. Thus the Katana Web Viewer was born, built into an Angular component with injected Unity WebGL content.





Unity WebGL

I first evaluated my options for building the web viewer. After exploring some prebuilt tools like Sketchfab, or Clara.io, it was concluded that a custom solution was needed for the functionality that we required. I built out the resultant web viewer using Unity's WebGL, utilizing Unity's robust capabilities alongside the proven nature of its technology. Due to my familiarity with Unity, I was able to quickly produce the web viewer with minimal loss.

JHipster and Angular

JHipster boilerplate code

Katana was already an existing JHipster web application

Next, I had to familiarize myself with the existing web app that was being built using JHipster while I was simultaneously developing Katana XR. JHipster runs Angular on the front-end, allowing me to quickly delve in with technology that I was already familiar with. By coordinating with the web application developers, I was able to integrate seamlessly into my colleagues' workflow, allowing me to contribute to another team's project without interrupting their development.

Dependency Injected Content

By converting the resultant Unity WebGL JavaScript files into exported modules, I was able to successfully inject the Unity content into a custom Angular component, allowing it to be called anywhere in the application. This included a custom JavaScript plugin that was implemented on the Unity side that allowed the passing of data back and forth from the Angular to the Unity app. In the end, I was able to procedurally process and load 3D content in the form of .obj files that had been previously uploaded through the Angular app.
