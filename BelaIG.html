Bering Land Bridge National Preserve Interactive Game

While working as a contractor for Humulo Engineering, I was tasked with creating a web based application for the US Department of Interior. The goal was to create an interactive game that allowed users to learn about the flora and fauna of the Bering Land Bridge National Preserve while presenting an accurate representation of the geographic features in all four seasons. The target audience was children K-6, with and emphasis on exploration and education. The project was constructed using Unity WebGL, and was built to be mobile first and as lightweight as possible.
