Princess Lily

Blender, Photoshop, Unity3D

The character was created from client specifications and source imagery. Several rounds of client feedback and critiques helped shape the model to the final look and feel.

Lip-Syncing

It was important to create an entity that was fully lip-synced since the character's face would be prominently visible and interactive. During modeling, phonemes were added to the 3D object alongside lip-sync plugins that were integrated engine side. The character was mapped to the provided voice clip, and paired with an appropriate body-language animation for the final product.

Base Model

Character additions such as wings, particles, and flowers were added in Unity, in order to simplify the model and provide more version control on certain aspects. The final character came out to be 5,231 triangles, 3,183 vertices, and 2,851 faces.
